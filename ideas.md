# Ideas

## General Features

- **Undo / Redo** – Track canvas actions (add, move, delete, connect) and allow stepping back and forward
- **Preset system** – Save and load named canvas snapshots locally (localStorage) alongside the existing URL sharing
- **MIDI input** – Map an external MIDI controller/keyboard to synth or drum modules for live play
- **Tap tempo** – Tap a button to set BPM by feel instead of using the slider
- **Module copy / paste** – Duplicate a module with all its current parameter values
- **Dark / light theme toggle** – Alternate canvas color scheme
- **Recording / export** – Bounce the master output to a WAV or WebM file via MediaRecorder
- **Keyboard shortcuts** – Space for play/stop, Delete for removing selected modules/cables, etc.
- **Mobile-friendly layout** – Responsive touch controls and simplified toolbar for small screens

## Modules

- **Filter** – Low-pass / high-pass / band-pass filter with cutoff frequency and resonance knobs; insert between any sound source and the output
- **Reverb** – Convolution or algorithmic reverb with decay time and wet/dry mix for spatial depth
- **LFO** – Low-frequency oscillator that modulates another module's parameter (e.g. filter cutoff, synth volume) for movement and wobble
- **Sampler** – Load a short audio sample (drag-and-drop or file picker) and trigger it from a step sequencer
- **Mixer** – Multi-input channel strip that sums several sources with individual volume and pan before hitting the output
- **Distortion** – Waveshaper / overdrive effect with drive amount and tone control
- **Compressor** – Dynamic range compressor with threshold, ratio, attack, and release to glue the mix together
- **Oscilloscope** – Visual-only module that displays the incoming audio waveform in real time on a small canvas

## Fun Modules

- **Gato 🐱** – A cute animated cat that meows, purrs, hisses, or chirps on each trigger. Pick the sound type and pitch, sequence it like a percussion instrument, and watch the cat react with matching animations
- **Duck Choir 🦆** – A row of rubber ducks that quack in harmony. Assign each duck a pitch and build chords; ducks bob and squish when they sound off
- **8-Bit Jukebox 🕹️** – Retro chiptune generator with tiny pixel-art visualizer. Choose classic game-sound presets (coin, jump, power-up, game-over) and sequence them
- **Theremin Ghost 👻** – A wobbly ghost that follows the mouse cursor; X controls pitch, Y controls volume. Spooky sine-wave sounds with cartoon ghost expressions
- **Beatbox Mouth 👄** – Animated mouth that lip-syncs to kick, snare, hi-hat, and bass sounds. Step-sequence the mouth and watch it beatbox
- **Magic Wand ✨** – Wave a sparkle trail across the module to trigger ascending/descending arpeggios and shimmer effects. Pure fairy-tale energy
